2. คำนวณเกรด

BEGIN

INPUT score

IF score >= 80
    grade = A
ELSE IF score >= 70
    grade = B
ELSE IF score >= 60
    grade = C
ELSE IF score >= 50
    grade = D
ELSE
    grade = F

OUTPUT grade

END

3. หาค่าสูงสุดของตัวเลข 3 จำนวน

BEGIN

INPUT number1
INPUT number2
INPUT number3

max = number1

IF number2 > max
    max = number2

IF number3 > max
    max = number3

OUTPUT max

END
